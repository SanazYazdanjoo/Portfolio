
# Clean Portfolio — Responsive Minimal Starter

Responsive, professional baseline with:
- Mobile-first header (hamburger) + desktop nav
- Fluid container padding and scalable typography
- Responsive grid utilities ready

## Run
```bash
npm install
npm run dev
```

## Customize
- Edit `src/pages/*` to add content
- Edit `src/App.jsx` to modify nav or footer
- `src/styles/theme.css` has light/dark-ready CSS variables
- Tailwind is fully enabled
